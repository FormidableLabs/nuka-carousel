Contributing
============

## Install

```sh
$ yarn
```

## Build source

Build the publish Babel sources:

```sh
# One time build
$ yarn run build

# Watch
$ yarn run build-watch
```

## Running the demo

Run the webpack-dev-server on http://localhost:8080:

```sh
$ yarn start
```

## Tests

```sh
# Everything
$ yarn run check

# ... which really runs
$ yarn run lint
$ yarn run test
```
